1. Кіріспе

ONAIR Studio қосымшасын пайдаланған кезде сіздің жеке деректеріңіздің қорғалуы біз үшін өте маңызды. Осы саясат қандай деректер жиналатынын, қалай қолданылатынын және қалай қорғалатынын түсіндіреді.

2. Жиналатын деректер

Біз мына деректерді жинаймыз:

  • Аты-жөні — тіркелу кезінде
  • Электрондық пошта — аутентификация үшін
  • Телефон нөмірі — байланыс және SMS хабарламалар үшін
  • Брондау тарихы — жазылымдарды басқару үшін
  • Push-токен — хабарламалар жіберу үшін
3. Деректерді пайдалану

Жиналған деректер мына мақсаттарда қолданылады:

  • Студияға жазылымды растау және басқару
  • Push-хабарламалар мен SMS жіберу
  • Kaspi Pay арқылы төлемдерді өңдеу
  • Қолдау қызметін ұсыну
4. Үшінші тарап қызметтері

Қосымша мына сыртқы сервистерді пайдаланады:

🔥 Firebase (Google) 💳 Kaspi Pay 📊 Google Sheets 🤖 Gemini AI

Бұл сервистердің өздерінің құпиялылық саясаттары бар. Олар тек қосымшаның жұмысын қамтамасыз ету үшін пайдаланылады.

5. Деректерді сақтау

Деректер Firebase (Google Cloud) серверлерінде сақталады. Деректер шифрланған қосылым (HTTPS/TLS) арқылы беріледі. Firestore қауіпсіздік ережелері (RLS) тек авторизацияланған пайдаланушыларға қол жеткізуге мүмкіндік береді.

  • Деректерге қол жеткізу

1. Введение

При использовании приложения ONAIR Studio защита ваших персональных данных является для нас приоритетом. Настоящая политика объясняет, какие данные собираются, как они используются и как защищаются.

2. Собираемые данные

Мы собираем следующие данные:

  • Имя — при регистрации
  • Электронная почта — для аутентификации
  • Номер телефона — для связи и SMS-уведомлений
  • История бронирований — для управления записями
  • Push-токен — для отправки уведомлений
3. Использование данных

Собранные данные используются для:

  • Подтверждения и управления бронированиями студии
  • Отправки push-уведомлений и SMS
  • Обработки платежей через Kaspi Pay
  • Предоставления поддержки
4. Сторонние сервисы

Приложение использует следующие внешние сервисы:

🔥 Firebase (Google) 💳 Kaspi Pay 📊 Google Sheets 🤖 Gemini AI

Каждый из этих сервисов имеет собственную политику конфиденциальности и используется исключительно для обеспечения работы приложения.

5. Хранение данных

Данные хранятся на серверах Firebase (Google Cloud). Передача данных осуществляется по зашифрованному соединению (HTTPS/TLS). Правила безопасности Firestore (RLS) обеспечивают доступ только авторизованным пользователям.
